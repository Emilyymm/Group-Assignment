HTML, CSS, and JavaScript

This project is a hands-on exercise in building a small website while applying lessons from class. We focused on semantic HTML, modular CSS, responsive layout techniques like Grid and Flexbox, accessible forms, and basic JavaScript for interactivity.

HTML gives the structure, CSS controls layout and appearance, and JavaScript adds behavior. Organizing files and using clear naming conventions made teamwork easier and debugging faster.
